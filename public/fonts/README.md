# Fonts

## PretendardVariable.woff2

This directory must contain `PretendardVariable.woff2`. It is loaded by
`src/lib/fonts.ts` via `next/font/local` (weight range `45 920`).

The bootstrap script downloads Pretendard **1.3.9** here automatically. To
update or restore the file manually:

1. Visit https://github.com/orioncactus/pretendard/releases
2. Download the latest `Pretendard-*.zip`
3. Extract and copy `dist/web/variable/PretendardVariable.woff2` to this folder
4. Keep the filename exactly `PretendardVariable.woff2`
