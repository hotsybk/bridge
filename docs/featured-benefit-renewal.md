## 3. Section 2 — 혜택 카드 진단

### 3.1 현재 상태 (스크린샷 2)

```

│           │ │           │ │           │ │           │
│ 5% 단일   │ │ 3일 만에  │ │ 매달 자동 │ │ 모이면    │
```

### 3.2 문제점 3가지

1. **카드 상하 분리 어색** — 텍스트 영역 + 그라데이션 + 아이콘 영역이 3-부분 구조 → "왜 분리됐지?" 의문
2. **색상 통일성 부족** — 4개 카드의 그라데이션이 회색·녹색·주황·파랑 4가지 → 4 색상 = 5색 (수수료 accent 포함) 너무 많음
3. **아이콘만 — 의미 약함** — 큰 TrendingDown / Wallet / Repeat / Users 아이콘. 의미 전달은 OK 지만 Apple 처럼 product photo 가 들어가면 더 풍부

### 3.3 Apple 혜택 카드 (참고, 이미지 3장째)

```
┌────────────────┐
│ 할부 결제   [+] │
│ 최대 18개월     │
│ 무이자 할부 결제 │
│                 │
│ 제휴 신용카드를  │
│ 사용해 자신에게  │
│ 알맞은 결제      │
│ 옵션을 선택해    │
│ 보세요.          │
│                 │
│                 │
│   [Apple Card  │  ← 카드 사진 부분 cropping
│    검은색 카드]  │
│   ··· 1234      │
└────────────────┘
```

특징:
- 흰 카드
- 텍스트 (헤드라인 + 부제)
- **하단에 product 사진** (카드·시계·이어팟·쇼핑백) — 카드 안에서 부분 cropping
- 통일된 흰 배경, accent 색 사용 안 함
- [+] 버튼 우상단

## 4. Section 2 리뉴얼 방향

### 4.1 적용 변경 4가지

| # | 변경 | 효과 |
|---|---|---|
| 1 | 4개 카드 그라데이션 색상 통일 — 모두 accent-light 또는 회색 | 통일성 ↑ |
| 2 | 큰 아이콘 → 의료품 이미지 또는 추상 그래픽 | Apple 풍 |
| 3 | 아이콘 + 그래픽 합성 (아이콘 작게 + 배경 추상 도형) | 깊이감 |
| 4 | 우상단 [+] 버튼 유지 | Apple 패턴 동일 |

### 4.2 대안 비교

**옵션 A — Apple 동일 (Product 사진)**:
- 결제: 신용카드 / 결제 단말 사진
- 정산: 지폐 / 시계 사진
- 자동: 캘린더 / 자동 화살표 일러스트
- 공동구매: 병원 외관 / 사람 손 모음

장점: Apple 그대로
단점: 의료 도메인에 맞는 이미지 찾기 어려움 (특히 정산·자동) + 무관한 이미지 사용 시 어색

**옵션 B — 추상 그래픽 + 아이콘 작게** (추천):
- 카드 배경 = 통일된 옅은 그라데이션 (accent-light → white)
- 하단 1/3 에 큰 추상 도형 (원·곡선·차트 등) + 그 위에 Lucide 아이콘 작게
- 4개 카드 동일 색상 톤

예시:
```
┌────────────────┐
│ 결제      [+]  │
│ 5% 단일 수수료  │
│ 거래 규모 커지면 │
│ 더 낮아집니다.   │
│                 │
│      ┌──┐       │  ← 작은 큐브·차트·그래픽
│     ╱ % ╲       │     accent 색
│    └────┘       │
│                 │
│ accent-light    │  ← 옅은 그라데이션 배경
│ gradient        │
└────────────────┘
```

장점: 의료 무관 이미지 회피 + 통일감
단점: 추상 — 의미 전달 약함 (그래도 헤드라인이 충분히 전달)

**옵션 C — 현재 유지 + 색상만 통일** (가장 단순):
- 큰 Lucide 아이콘 유지
- 4개 카드 모두 같은 그라데이션 (accent-light → transparent)
- 아이콘 색상 accent 통일

장점: 가장 빠름
단점: Apple 풍이라고 보긴 어려움

### 4.3 추천

**옵션 B (추상 그래픽 + 아이콘)** 권장.

이유:
- Apple 의 사진 접근은 우리 도메인에 안 맞음 (의료 카드/시계/이어팟 같은 universal product 없음)
- 단순 색상 통일은 Apple 풍 부족
- 추상 그래픽 + 작은 아이콘 = 디자인 톤 ↑ + 의미 전달 OK + 도메인 무관
